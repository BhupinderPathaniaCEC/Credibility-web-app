### 1. Environment Setup
- [ ] Install .NET 10 SDK and Node.js (Latest LTS).
- [ ] Clone the repository and open CredibilityIndex.sln.
- [ ] Ensure appsettings.Development.json in the API project points to your local SQL instance.

### 2. Running the Backend (API)
- [ ] Navigate to src/CredibilityIndex.Api.
- [ ] Run dotnet run or press F5 in VS Code.
- [ ] Verify Swagger UI loads at https://localhost:[port]/swagger.

### 3. Running the Frontend (UI)
- [ ] Navigate to ui/credibility-ui.
- [ ] Run npm install.
- [ ] Run npm start (or ng serve).
- [ ] Access the app at http://localhost:4200.
